## Building Process

1. Install all the dependencies - `npm install`
2. build the javascript files - `npm run build`
3. build the package
    Windows - `npm run package:win`
    Linux - `npm run package:linux`
    Mac - `npm run package:mac`
    
4. run the package - `./build/cowin-checker`
